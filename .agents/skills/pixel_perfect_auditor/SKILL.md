# Skill: Pixel Perfect Auditor

Этот скилл предназначен для проведения глубокого визуального анализа интерфейса.

## Основные задачи
1. **Cross-Device Check**: Проверка верстки на разрешениях Desktop (1920x1080), Tablet (iPad Air), и Mobile (iPhone 14).
2. **Visual Consistency**: Проверка единства шрифтов, отступов (padding/margin) и цветовой палитры.
3. **Motion Audit**: Проверка плавности анимаций GSAP и Lenis.

## Инструменты
- `browser_subagent`: Основной инструмент для захвата скриншотов.
- `capture_browser_screenshot`: Должен вызываться с четким описанием вьюпорта.

## Алгоритм работы
1. Открыть тестируемую страницу на Vercel.
2. Сделать серию скриншотов:
   - Вся страница (full page) для анализа структуры.
   - Hero-секция (fold) для анализа первого впечатления.
   - Футер и мобильное меню.
3. Сравнить полученные изображения с оригиналом `vittorio-parfum.ru`.
4. В отчете указать:
   - Баги верстки.
   - Отклонения в типографике.
   - Проблемы с читаемостью текста на фонах.

## Фокус на деталях (Premium)
- Отсутствие "лесенки" в анимациях.
- Отсутствие горизонтального скролла на мобильных устройствах.
- Качество изображений (отсутствие артефактов сжатия).
