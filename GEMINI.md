# Project GEMINI: Vittorio Parfum Clone (Next.js)

## 1. Основные правила и ограничения
- **Рабочая директория**: `/home/usic/.gemini/antigravity/scratch/www`
- **СТРОГАЯ ИЗОЛЯЦИЯ**: 
  - Работаем ТОЛЬКО внутри папки `www`.
  - ЗАПРЕЩЕНО читать, индексировать или анализировать файлы из соседних папок (`../vittorio-web`, `../Velveto-Team`, `../analytics`).
  - Это критически важно для предотвращения переполнения памяти и зависаний.
- **Цель**: Создать точную копию (Pixel Perfect) сайта `https://vittorio-parfum.ru/` и улучшить его.
- **NO EXTERNAL LINKS**: Строго использовать только локальные ресурсы (public/). Запрещены прямые HTML-ссылки на домен-донор.
- **DEPLOYMENT READY**: Код должен быть готов к передаче заказчику (чистый, структурированный, без хардкода).
- **ТЕСТИРОВАНИЕ**: Не использовать локальный сервер (`npm run dev`) и локальный браузер для проверки верстки, если не оговорено иное. Заливать изменения и проверять всё сразу на Vercel.
- **САЙТ-ВИЗИТКА**: Проект является исключительно имиджевым сайтом-визиткой (portfolio/showcase). Функционал интернет-магазина (корзина, чекаут, онлайн-покупка) НЕ реализуется. Все кнопки вроде "Оформить заказ" ведут на Контакты или внешние ссылки-мессенджеры.

## 2. Технологический стек
- **Framework**: Next.js (React)
- **Стилизация**: CSS Modules / SCSS (избегать Tailwind, если не оговорено иное, для максимального контроля Pixel Perfect).
- **Язык**: TypeScript
- **Управление состоянием**: React Context / Zustand (по необходимости).
- **Анимации**: GSAP / Lenis (для плавного скролла и премиальных эффектов).

## 3. Инструменты и Ресурсы
- **Nano Banana**: Генерация и редактирование изображений.
- **Veo 3**: Генерация видео-контента (DeepMind).
- **GitHub**: Контроль версий.

## 4. Этапы разработки
1. **Инициализация**: Настройка чистого Next.js проекта.
2. **Структура**: Создание каркаса страниц (Home, Catalog, Product, About, etc.).
3. **Assets**: Подготовка графики и видео с помощью Nano Banana и Veo 3.
4. **Верстка**: Pixel Perfect реализация каждого блока.
5. **Оптимизация**: Lazy loading, SEO, Performance.

## 7. Custom Agent Skills
В проекте реализованы специализированные «навыки» агента для повышения качества. Перед выполнением специфических задач агент должен обращаться к инструкциям в папке `.agents/skills/`:
- **Pixel Perfect Auditor**: Проверка визуального соответствия и адаптивности.
- **Luxury Content Lab**: Инструкции по генерации премиум-медиа (Nano Banana, Veo 3).
- **Localization Guard**: Контроль качества переводов и целостности JSON-файлов.

## 8. ДИЗАЙН ФЛАКОНОВ (КРИТИЧЕСКОЕ ПРАВИЛО)
**СТРОЖАЙШИЙ ЗАПРЕТ НА ВЫДУМАННЫЕ ДИЗАЙНЫ.** Использовать только оригинальный дизайн флакона Vittorio, задокументированный на исходных фотографиях от клиента. При любых генерациях изображений или 3D-моделировании строго соблюдать следующие параметры:
- **ФОРМА**: Плоская прозрачная стеклянная фляга (flat flask / eye-shaped base). Абсолютно гладкая, без единого острого угла, полностью скругленная (овал). Широкий фасад, узкий профиль.
- **СТЕКЛО**: Прозрачное, темное коричнево-янтарное (дымчатое). Толстое стеклянное дно, создающее преломление света.
- **КРЫШКА**: Гладкий серебристый металлический цилиндр. Важная деталь: у основания крышки есть "ступенька" (широкое кольцо у горлышка, затем узкий перешеек, затем сам широкий цилиндр крышки).
- **ЗАПРЕЩЕНО**: Квадратные, круглые, шарообразные формы. Граненые "кристальные" дизайны. Матовое или непрозрачное стекло. Рифленые крышки. Любое отклонение от этого правила считается грубой ошибкой.
